# Vice City - Theme for Omarchy

## To install:
1. Copy `https://github.com/lavarinimoreira/omarchy-vice-city-theme`
2. Click `SUPER + ALT + SPACE`
3. Click `Install`
4. Click `Theme`
5. Paste the link

### 2nd way to install
```bash
omarchy-theme-install https://github.com/lavarinimoreira/omarchy-vice-city-theme
```
